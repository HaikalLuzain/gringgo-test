# Gringgo Test

This repo contains the CRUD for provinces.

## Local Environment
For development, I suggest you pre-install the following:

1. [Node js & NPM (Latest)](https://nodejs.org/en/download/)

## Usage

### **Install**
Download all dependencies

```bash
npm install 
or 
yarn install
```

### **Run Local Development**

```bash
npm run start
or
yarn start
```
